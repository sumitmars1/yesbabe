我将修改 `src/views/Chat/Component/MessageCards/parts/ImageContent.vue` 文件，以优化非 PRO 用户的图片显示过渡效果。

### 问题分析

当前实现中，当图片加载完成时，“模型封面图”（占位图）会被立即替换为生成图片的“马赛克”效果。由于两张图片内容完全不同，这种瞬间切换会造成视觉上的突兀感。

### 解决方案

我将实现一个**平滑的淡出过渡效果**。

1. **重构占位层**：将原本在 `n-image` 内部插槽中的“封面图”和“加载转圈”逻辑提取出来，作为一个独立的遮罩层（Overlay）覆盖在图片上方。
2. **分离状态控制**：

   * **加载状态（Spinner）**：图片加载完成后立即隐藏。

   * **封面图背景**：在图片加载完成且马赛克绘制完毕后，通过 CSS `transition` 平滑淡出（如 0.5秒），而不是立即消失。
3. **最终效果**：用户将看到封面图慢慢淡去，平滑地显露出下方的马赛克内容，消除闪烁和突兀感。

### 具体实施

* 新增 `showTransitionCover` 响应式变量控制封面图显示。

* 调整 DOM 结构，确保遮罩层 `z-index` 位于马赛克层之上，但低于“解锁按钮”层。

* 添加 CSS `transition: opacity 0.5s ease-in-out` 实现柔和过渡。
