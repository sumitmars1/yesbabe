# 1：每个角色的历史消息记录接口格式
```json
{
    "id": "number",
    "type": "text", //我发送的消息类型，现在指定为text，可扩展
    "content": "string", //我发送的消息
    "additional_data": "string",
    "is_processed_by_third_party": "boolean",
    "created_at":"string",
    "updated_at": "string",
    "response_id": "number",
    "response_type": "image" | "text" | "video" | "mixed", //AI回复的消息类型，现在指定为image、text、video、mixed可扩展
    "response_content": "string", //AI回复的消息 根据response_type不同，内容不同，image和video是通过接口获取，text和mixed是直接返回，mixed可能为图片或者视频，根据尾缀的类型判断
    "response_additional_data": "string",
    "response_is_from_third_party": "boolean",
    "response_created_at": "string"
}
```
